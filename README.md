# Aetrex Scanner Fleet Management

Vite + React app for the Aetrex Scanner Fleet Management Control Panel.

## Stack

- Vite + React
- Tailwind CSS v4
- React Router
- Chart.js (`react-chartjs-2`)
- Leaflet (`react-leaflet`)
- DataTables
- Papa Parse (CSV map data)
- Font Awesome

## Getting started

```bash
npm install
npm run dev
```
